# Mini ERP + CRM Operations Portal

A full-stack ERP/CRM operations portal designed for a wholesale and distribution business.

## Project Status

🚧 Currently under active development.

## Tech Stack

### Frontend
- React.js
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- TypeScript

### Database
- PostgreSQL
- Prisma ORM

### Authentication
- JWT
- Role-Based Access Control

## Core Modules

- Authentication & Role Management
- Customer CRM
- Product & Inventory Management
- Stock Movement Tracking
- Sales Challan Management
- Dashboard

## Roles

- Admin
- Sales
- Warehouse
- Accounts
